# پوشه Object Files

این پوشه شامل فایل‌های موقت و میانی کامپایل است.

## محتویات:

- **فایل‌های .obj**: فایل‌های شیء کامپایل شده
- **فایل‌های .pdb**: فایل‌های اطلاعات دیباگ
- **فایل‌های موقت**: فایل‌های موقت کامپایلر

## نکات مهم:

- این پوشه به صورت خودکار در زمان build ایجاد می‌شود
- فایل‌های موجود در این پوشه موقت هستند
- می‌توانید این پوشه را پاک کنید و دوباره build کنید
- این فایل‌ها برای دیباگ و بهینه‌سازی استفاده می‌شوند
